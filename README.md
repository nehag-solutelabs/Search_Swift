# DispatchWorkItem #

- DispatchWorkItem is used to store a task on a dispatch queue for later use and you can perform operations serval operations on it, you can even cancel the task if it is not required later in the code.
- Primarily used in scenarios where we require the capability of delaying or canceling a block of code from executing.
- It has ability to wait until the work done.

## Methods ##

- perform()
- notify()
- wait()
- cancel()
